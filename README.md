# GitHub Pages Site

This repository includes a GitHub Pages site scaffold. To enable GitHub Pages:

1. Go to the repository settings.
2. Scroll down to the "GitHub Pages" section.
3. Select the source to be the `main` branch and the `/docs` folder.

Your site will be published at `https://<username>.github.io/<repository>/`, where `<username>` is your GitHub username and `<repository>` is the name of this repository.
